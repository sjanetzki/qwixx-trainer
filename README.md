# qwixx-trainer
A genetic algorithm that learns to play the game Qwixx
## Setup
install [Python](https://www.python.org/downloads/)
go to cmd:
```shell
git clone https://github.com/sjanetzki/qwixx-trainer
cd quixx_trainer
pip install -r requirements.txt
```

## Usage
### Trainer
--> genetic algorithm
```shell
python trainer.py
```

### Game
--> play via PyGameUi
```shell
python game.py
```
